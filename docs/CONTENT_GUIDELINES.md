# EduEvidence JP — Content Editorial Guidelines

このドキュメントは、新しい strategy(指導法ページ)・column(コラム)を追加する際の **編集ポリシーと品質基準** を定めます。エビデンスベースのサイトとして守るべき最重要ルールです。

---

## 0. なぜこのドキュメントがあるか

運営初期、AI 支援で下書きを生成した際、**事前に文献を検索せずに効果量を「常識的な範囲で」割り当てた** ことが原因で、後の検証で複数の数値が研究と一致しないことが判明しました(例: 朝食欠食を「+3ヶ月」と書いたが、実際の研究は「効果は subtle で月数換算なし」)。

エビデンスベースのサイトにおいて、**根拠のない数値を権威ある形式で提示すること** は、最も避けるべき不誠実です。教員が「研究で確かめられた数字」だと信じて読むからです。このドキュメントは、その失敗を二度と繰り返さないための歯止めです。

---

## 1. 絶対ルール(Non-negotiable)

### Rule 1.1 — `monthsGained` は研究で裏付けがある時だけ非ゼロを書く

**根拠の無い数値を書かない**。「常識的にこれくらいだろう」「他の指導法と比較してこれくらいかな」という推測は禁止。

| 状況 | 設定すべき値 |
|---|---|
| メタ分析・RCT で具体的な効果量(d, g)が報告されている | その値を月数換算した数(d=0.1 ≈ 1ヶ月) |
| 観察研究のみで効果量が%表記など換算困難 | **1**(控えめ) + 注記で「観察研究のみ」と明示 |
| 学力への効果を測った研究が存在しない | **0**(未証明) + 注記で「学力 RCT は存在しない」と明示 |
| 効果が負(逆効果)の研究が報告されている | 負の値(例: -2) + 注記 |

### Rule 1.1a — 出典階層

`monthsGained` の **代表値** は次の優先順位で決める:

1. **日本研究(信頼度 ★★★ 以上)** があればそれを採用(文化的妥当性が最も高い)
2. **EEF Toolkit** に該当項目があれば EEF を採用(方法論が最も厳格、継続更新)
3. **Hattie の Visible Learning** しか無ければ参考値として使うが、**必ず `evidence.hattie` フィールドに記録し、注記で「楽観値の傾向」を明示**
4. いずれも無ければ **0(未証明)**

### Rule 1.1b — 併記と文化的注記

可能な限り、`evidence.eef` / `evidence.japan` / `evidence.hattie` を併記し、読者が比較できるようにする。両者で値が大きく異なる場合は `culturalContext` フィールドに **なぜ差があるか(文化的・制度的要因)** を説明する。

frontmatter 例:
```yaml
monthsGained: 2  # 代表値 = 日本研究優先(EEF は +5 だが日本文脈では +2)
evidenceStrength: 3
source: mixed    # eef / japan / hattie / mixed の 4 値
evidence:
  eef:
    monthsGained: 5
    strength: 4
    note: "英国での効果。宿題と学力の関係をメタ分析で検証"
  japan:
    monthsGained: 2
    strength: 3
    note: "日本では既に宿題量が多いため追加効果は限定的"
    researcher: "中室牧子"
culturalContext: |
  日本の家庭学習時間は既に国際的に長い水準にあるため、EEF の +5 ヶ月をそのまま期待できない可能性がある。量を増やすより質(授業との連動・フィードバック)に投資する方が効果的。
```

### Rule 1.2 — 引用論文の citation は必ず一次情報で確認する

WebSearch / WebFetch で **論文の存在・著者・年・タイトル・ジャーナル・巻号・ページ** を検証してから掲載する。

過去の失敗例:
- King et al. 2005 を「Behavior Modification」誌と書いた → 実際は「Behavioral Disorders」誌(完全に違うジャーナル)
- Skaalvik & Skaalvik 2017 を「vol 67」と書いた → 実際は「vol 65」
- Finland PISA 2018 を「507」と書いた → 実際は「503」

これらは「ありそうな数字」「ありそうな雑誌名」を記憶ベースで書いたために発生しました。

### Rule 1.3 — DOI は必ずアクセスして 200 が返ることを確認する

リンク切れは GitHub Actions で自動検知しますが(§7 参照)、執筆時点でも DOI が開けることを手動で確認してください。

### Rule 1.4 — 「学力以外の効果」を「学力効果」と混同しない

教員の働き方改革・不登校支援・SEL のように、**第一の価値が学力ではない政策**については、`monthsGained` に学力換算値を書かない。0 にして注記で別の価値を説明する。

### Rule 1.5 — 画像・動画の引用は原則としてリンク/公式埋め込みに限定する

ゲーム・アプリ・商業コンテンツの画面を記事に埋め込むと、著作権法 32 条の引用要件(主従関係・引用の必然性・出所明示・無改変)を満たすのが難しくなります。EduEvidence は一次研究の引用で信頼性を担保するサイトであり、画像引用の曖昧さでブランドを損なうリスクは避けるべきです。

**採用してよい方式:**

1. **公式ページへの通常リンク** — Astro Markdown のリンクで十分。読者はクリックで公式画面に到達できる
2. **公式提供の埋め込みコード** — YouTube 公式トレーラー等、埋め込み機能が許諾と一体になっているもの
3. **報道向けプレスキット画像** — 明示的に許諾されている画像のみ採用し、出所を明記
4. **自前のイラスト・図解** — 完全なオリジナル

**避ける方式:**

- ニュース記事掲載 screenshot の転載(二次著作物、出所が不明瞭)
- ゲーム画面を自分で撮影して直接掲載(主従関係を崩しやすい)
- SNS 投稿画像を画像として保存・掲載

**厳格な 32 条引用が成立しうる例外:**

- 画像そのものが批評の対象である(例: UI デザインを批判する記事の該当 UI)
- 本文の分析量が画像を明確に上回っている(主従関係)
- 出所・著作権者を明示し、無改変で提示する

この例外に該当するかは個別判断が必要です。迷ったら 1〜4 に退避してください。

### Rule 1.6 — 学校と家庭の役割境界を明示する

子どもを取り巻く問題(デジタル・睡眠・食事・家族関係等)には、**学校が担える領域** と **家庭・地域社会が担うべき領域** が並行して存在する。コラム・戦略ページでこれらのトピックを扱うときは、**責任の線引き** を読者に伝わる形で示す必要がある。

日本の教育言説には、「家庭・地域で起きた問題も含めて教師が対応する」という暗黙の包括責任観がしばしば見られる。EduEvidence のコンテンツは、**教師が何でも屋ではない** という立場を明確にし、その上で学校の専門的役割がどこにあるかを記述する。

**避けるべき書き方:**

- 家庭で起きた問題の解決策として「教室でできる ○○」だけを提示する
- 「学校が ○○ すれば解決する」という論調で、家庭・地域の役割を黙殺する
- SEL・行動介入などの学校向けエビデンスを、家庭責任の代替手段として提示する

**推奨される書き方:**

- まず「どこで起きている問題か」「誰が第一義的な責任を持つか」を明示する
- その上で「学校にできる役割はどこまでか」を定義する
- 家庭を支える学校の働きかけ(情報提供・保護者会)と、学級内で起きた時の対応(SEL・行動介入)を分けて記述する
- 海外の教育制度との対比を持ち出す場合、「役割分担の違い」を含めて公平に扱う

### Rule 1.7 — 「チーム学校」の枠組みで組織対応を示す(教員個人で抱えない)

Rule 1.6 が「家庭・地域と学校の境界」を扱うのに対し、本ルールは **学校の内側での役割分担** と **学校と関係機関との連携** を扱う。

不登校・いじめ・特別支援・家族関係・貧困など、担任個人の努力だけで対応できない問題領域では、**「チーム学校」** の枠組みで組織的に対応することが前提となる。「チーム学校」概念は [2015 年の中央教育審議会答申(中教審第 185 号)](https://www.mext.go.jp/b_menu/shingi/chukyo/chukyo0/toushin/1365657.htm) で提示され、[文部科学省『生徒指導提要(改訂版)』(2022)](https://www.mext.go.jp/a_menu/shotou/seitoshidou/1404008_00001.htm) で運用面が体系化、[2024 年の中央教育審議会答申(中教審第 251 号)](https://www.mext.go.jp/b_menu/shingi/chukyo/chukyo3/079/sonota/1412985_00006.htm) で専門スタッフ配置の拡充を含む最新の政策方針が示された。

EduEvidence のコラム・戦略ページがこれらの領域を扱うとき、担任個人の実践だけに閉じず、公式文書で使われる次の 3 層を使い分けて記述する。

| 層 | 公式用語 | 主な構成 |
|---|---|---|
| 校内の教職員 | **教職員** | 担任、養護教諭、特別支援教育コーディネーター、生徒指導主事、管理職 等 |
| 校内の専門スタッフ | **専門スタッフ** / **専門的人材** | スクールカウンセラー(SC)、スクールソーシャルワーカー(SSW)、ICT 支援員 等 |
| 学校外の連携先 | **関係機関** | 児童相談所、医療機関、市町村福祉部局、警察、要保護児童対策地域協議会、教育支援センター、フリースクール、NPO 等 |

生徒指導提要は、校内で機動的に動かす **校内連携型支援チーム** と、関係機関を含めた **ネットワーク型支援チーム** を区別して提示している。コラムで「連携」を書くときは、どの層を指しているかが読者に伝わる粒度で記述する。

**避けるべき書き方:**

- 担任・学級担任レベルの実践だけで問題解決を論じる
- 「担任の気づきが重要」「教員がもっと生徒を見るべき」という精神論で締めくくる
- 組織対応を前提とすべき法制度(いじめ防止対策推進法・教育機会確保法・発達障害者支援法等)を扱うのに、担任個人の責任として描く
- 「専門家」「外部機関」のような抽象語に留まり、どの専門スタッフ・どの関係機関かを具体化しない

**推奨される書き方:**

- 担任レベルの対応と、**専門スタッフ・関係機関** による対応を並列で記述する
- 関連法制度における **組織対応の位置付け**(いじめ防止対策推進法の「いじめの防止等のための対策の組織」、教育機会確保法の関係機関連携 等)を示す
- 教員の時間・人員に限界があることを前提とし、**関係機関や自治体レベルで補われるべき領域** との接続を示す
- 用語は公式文書(学習指導要領・生徒指導提要・関連答申)の呼称を優先し、恣意的な言い換えを避ける

**適用が想定される主なトピック:**

- 不登校(SC・SSW、教育支援センター、フリースクール)
- いじめ(いじめ防止対策推進法の組織対応、SC、場合により警察・医療機関)
- 特別支援(特別支援教育コーディネーター、通級指導教員、医療・療育機関)
- 共同親権・家族関係(SSW、スクールロイヤー、関係機関)
- 児童虐待(児童相談所、要保護児童対策地域協議会、SSW)
- 教育格差・貧困(SSW、市町村福祉部局、自治体の学習支援事業)

Rule 1.6 と Rule 1.7 は併存する概念。Rule 1.6 は **学校の外側との境界**、Rule 1.7 は **チーム学校の枠組みでの校内連携と関係機関との連携** を扱う。両者をセットで適用することで、「教員を何でも屋にしない」という編集姿勢が一貫する。

**主な参照文献:**

- 中央教育審議会 (2015). [チームとしての学校の在り方と今後の改善方策について(答申)](https://www.mext.go.jp/b_menu/shingi/chukyo/chukyo0/toushin/1365657.htm). 中教審第 185 号. — 「チーム学校」概念の原典。
- 文部科学省 (2022). [生徒指導提要(改訂版)](https://www.mext.go.jp/a_menu/shotou/seitoshidou/1404008_00001.htm). — 校内連携型支援チーム・ネットワーク型支援チームなど運用面の詳細。
- 中央教育審議会 (2024). [「令和の日本型学校教育」を担う質の高い教師の確保のための環境整備に関する総合的な方策について(答申)](https://www.mext.go.jp/b_menu/shingi/chukyo/chukyo3/079/sonota/1412985_00006.htm). 中教審第 251 号. — 2024 年 8 月 27 日。教職員集団の多様化・専門スタッフ配置拡充を含む最新の総合方策。

### Rule 1.8 — 主張の強度は原典を超えない(脚色禁止)

Rule 1.2 が **引用内容の事実性** を扱うのに対し、本ルールは **主張の強度の忠実性** を扱う。メタ分析・ガイダンスなどのソースが示す範囲を、執筆時に無意識に強めないこと。

**避けるべき書き方:**

- 「効果がある」とソースが言っているところを、「**特に大きな**効果がある」「**最も**効果が大きい」と書く
- 「A と B 双方に効く」を「A に**特に効く**」のように一方に強調を寄せる
- ソースが述べない因果方向を、「**決定的**」「**断然**」「**まさに**」のような副詞で補強する
- 単発の RCT を「メタ分析で証明」と書く、小規模研究を「広く支持されている」と書く

**過剰主張語の警戒リスト:**

- 最も / 圧倒的 / 特に大きな / 決定的 / 断然 / まさに / 他を引き離す / 飛び抜けて / 群を抜く
- これらを使いたくなった時は、原典の原文がこの強度を担保しているかを必ず確認する

**過去の失敗例(本サイト):**

- EEF の `"good teaching for SEND is good teaching for all"`(SEND への質の高い指導 = すべての子への質の高い指導)を「**SEND の子どもに特に大きな効果がある**」と書いていた。EEF は「特に大きな効果」とは述べておらず、SEND と全員の双方に資するという並列の主張 ([special-needs-8-8-percent.md](../src/content/columns/special-needs-8-8-percent.md), PR #42 で訂正)

**推奨される書き方:**

- ソースが「A は B につながる」と言うなら、コラムも「A は B につながる」と書く
- ソースの結論が曖昧・限定的なら、その限定を保って紹介する
- 「言えること」と「言えないこと」を分けて書く(エビデンスの限界を書くこと自体が本サイトの価値)

### Rule 1.9 — サイト内部の整合性を保つ

コラム・戦略ページ・About など、サイト内の記述同士が矛盾しないこと。

**整合を確認すべき対象:**

1. **数値の整合**: コラムで `/strategies/<slug>` をリンクしつつ効果量 `+N ヶ月` を書くとき、その値は戦略ページの `monthsGained` と一致すること([Issue #43](https://github.com/Hayato-Isagawa/edu-evidence/issues/43) で tracker 化)
2. **実践例の整合**: コラムが紹介する実践例が、リンク先の戦略ページの「注意したいこと」「形骸化の警告」と矛盾しないこと
3. **用語の整合**: 同一概念に同一の公式用語を用いる(例: 通常学級を「メインストリーム」と書かない、SC を「カウンセラー」と略記しない)
4. **役割分担の整合**: Rule 1.6 / 1.7 の記述が、複数コラムをまたいで一貫していること

**過去の失敗例(本サイト):**

- `bullying-recognition-paradox.md` が「朝の会で『今の気持ちを一言』」を SEL 的活動として推奨していたが、リンク先の [social-emotional-learning.md](../src/content/strategies/social-emotional-learning.md) は **「気持ちを言わせることが目的化すると形骸化する」** と明記していた(PR #42 で整合化)
- `special-needs-8-8-percent.md` が `one-to-one-tuition` の効果量を `+4 ヶ月` と記載していたが、戦略ページの `monthsGained` は `5` だった(PR #42 で訂正)

**推奨される運用:**

- コラムで戦略ページをリンクする時は、そのページを最低限読み、主要な主張・注意点と矛盾しない形で引用する
- 効果量を書くときは必ず戦略ページの `monthsGained` を確認する(将来的に自動チェックで補助: Issue #43)
- 複数コラムで同じ概念を扱う時は、既存の記述を調べてから書き、揃える

### Rule 1.10 — 実践例はソース記載または現場妥当性のあるものに限る

コラム・戦略ページの「教室でできる〇〇」「日本の小学校で取り入れるヒント」「まずできること」のような実践例セクションでは、**ソースに書かれていない「ありそうな助言」を、あたかもエビデンスから導かれた実践のように並べない**。本サイトの信頼は、一次研究の紹介と同時に、現場の教員が読んで違和感のない実装提案によって支えられている。

**採用可能な実践例の 3 段階:**

1. **ソース記載**: EEF ガイダンス・メタ分析・研究者の原典で **具体的に推奨されている** 実践
2. **公的文書記載**: 学習指導要領・生徒指導提要・関連答申の枠内にある実践
3. **執筆者の現場経験**: 元教員として実際に実践経験があり、効果を実感したもの — ただし「ソースの主張」と混同させない表記に留める(例: 「本サイトの執筆者の経験では」)

**避けるべき実践例パターン:**

- **儀式化しやすい実践** — 「朝の会で順番に気持ちを一言」「毎日全員が〇〇と言う」のような形骸化しやすい型(リンク先の戦略ページが形骸化を警告している場合は Rule 1.9 の違反でもある)
- **時間的に非現実的** — 「全員と週に 1 回 1 対 1」「毎日全員と 1 分」のような、35 人学級の日課に落ちない設計
- **実装できない一般論** — 「寄り添う」「受け止める」「大切にする」のような、具体的行動に落とせない語で止まる助言
- **戦略ページの警告と矛盾** — 「気持ちを言わせる目的化」を戒める SEL 戦略にリンクしながら、実践例にその形骸化パターンを挙げる
- **学年・発達段階の無視** — 小 1 と小 6 に同じ実践を並べ、発達段階による調整に触れない
- **教員の権限外の提案** — 学校組織や家庭の領域に属する判断を「担任ができること」として書く(Rule 1.6 / 1.7 の違反でもある)

**推奨される書き方:**

- ソース記載の実践は **そのまま紹介する**(例: 「EEF は足場かけとして〜を挙げている」)
- 公的文書の実践は **典拠を示して紹介する**(例: 「生徒指導提要は校内連携型支援チームによる〜を示している」)
- 現場経験ベースの実践は **「ソースとは別の補足」として明示する**(例: 「本サイトの執筆者の経験では、〜が機能しやすい」)
- 実装条件・例外・失敗パターンを併記する(「〜を目的化すると形骸化する」)
- 学年・発達段階による調整があれば、それを書く

**過去の失敗例(本サイト):**

- `social-emotional-learning.md` が「**朝の会で順番に気持ちを一言**」を取り入れるヒントとして挙げていた。ソースの EEF / Durlak に記載はなく、儀式化・形骸化しやすい型だった(削除済)
- `bullying-recognition-paradox.md` が「**毎日の SEL 的活動 — 朝の会で気持ち一言**」「**教師と子どもの 1 対 1 の時間 — 週に 1 分でも全員に声をかける**」を挙げていた。前者は SEL 戦略ページの「気持ちを言わせる目的化の警告」と矛盾(Rule 1.9)、後者は時間単位として意味不明だった(PR #42 で訂正)

## 2. 新しい strategy を追加する時のチェックリスト

新しい `src/content/strategies/*.md` を作る際、frontmatter を書く前に以下を完了する:

- [ ] **Step 1**: そのトピックに関するメタ分析・RCT を WebSearch で検索
- [ ] **Step 2**: 効果量(Cohen's d, Hedges' g, %)を一次資料で確認
- [ ] **Step 3**: 引用する論文を WebFetch で実在確認(著者・年・ジャーナル・タイトル)
- [ ] **Step 4**: 上記が満たせない場合、`monthsGained: 0` で書き、注記で「実証されていない」を明示
- [ ] **Step 5**: `evidenceStrength` も一次資料に基づく(★1: 観察のみ, ★2-3: 一部 RCT or 弱いメタ分析, ★4-5: 大規模メタ分析多数)
- [ ] **Step 6**: `cost` は実装時の現実的な負担(教員の工夫だけ=1、組織変革が必要=5)
- [ ] **Step 7**: 注記セクションに「根拠の質」「研究の限界」を必ず明記
- [ ] **Step 8**: 「日本の小学校で取り入れるヒント」等の実践例は、Rule 1.10 に従い **ソース記載 / 公的文書記載 / 執筆者の現場経験** のいずれかに限定し、儀式化しやすい型・非現実的な時間設計・一般論すぎる表現を避ける

### frontmatter の値の意味(再掲)

```yaml
monthsGained: 整数(-12〜12)。学力への因果効果。月数換算。0 は「未証明」を意味する
evidenceStrength: 1〜5。研究の質と量
  1: 観察研究のみ、質も量も限定的
  2: 弱い因果証拠、小規模 RCT or 観察研究
  3: 中規模のメタ分析あり、または複数の RCT
  4: 大規模メタ分析、複数の独立した RCT で再現
  5: 多数の大規模メタ分析、再現性が極めて高い
cost: 1〜5。導入コスト
  1: 教員の工夫だけで可能。予算ゼロ
  2: 少額の教材費、短時間の研修
  3: 中程度の研修、教材費、補助スタッフ
  4: 専門スタッフ配置、大規模研修
  5: 制度変更、組織再編、人員増
```

## 3. column(コラム)を追加する時のチェックリスト

- [ ] 引用するすべての論文・統計を一次資料で確認
- [ ] 数値(PISA スコア、効果量、率)は最新の公式データで照合
- [ ] 「○○年制定」「○○年施行」のような制度の年号は、両方確認(教育機会確保法は 2016 制定 / 2017 施行)
- [ ] 著名研究者(中室牧子・松岡亮二・Hattie・Willingham 等)の主張を引用する場合、原典のニュアンスから外れていないか確認
- [ ] 「~~は本当に効果があるのか?」のような問いを立てる場合、結論を出す前に **複数のメタ分析** で確認
- [ ] 「現場の声」「報告されている事例」を書く場合は、**一次ソース(記事・調査)へのリンクを本文または脚注で提示**する。出所なしの箇条書きは避ける(Rule 1.5 との整合)
- [ ] 画像・動画を紹介する場合は Rule 1.5 に従い、原則として公式ページへのリンクまたは公式提供の埋め込みで処理する
- [ ] 家庭・地域に起因するトピック(デジタル利用・睡眠・家族関係など)を扱う場合、Rule 1.6 に従い「家庭が起点 → 学校のサポート → 教室内対応」の順序で役割境界を明示する
- [ ] 不登校・いじめ・特別支援・家族関係・貧困など **担任個人で対応できない領域** を扱う場合、Rule 1.7 に従い「チーム学校」の枠組み(**教職員・専門スタッフ・関係機関**)での組織対応を、公式文書の用語を用いて明示する
- [ ] Rule 1.8 に従い、過剰主張語(「最も」「特に大きな」「決定的」等)を原典が担保するときだけ使う
- [ ] Rule 1.9 に従い、リンクする戦略ページの `monthsGained` と本文の効果量、リンク先の実践例・注意書きとの整合を確認する
- [ ] Rule 1.10 に従い、実践例は **ソース記載 / 公的文書記載 / 執筆者の現場経験** のいずれかに限定し、現場経験ベースのものは「ソースの主張」と混同させない表記にする
- [ ] §9 の表現ガイド(翻訳調・ビジネス用語を避ける)に沿って文章を推敲する
- [ ] §9「一次ソース特有の比喩・業界用語は要チェック」に従い、調査報告・メディア記事から引用する比喩的表現が一般読者に通じるかを確認する

### lastVerified の運用

- 初回公開時は `lastVerified` を設定しない(公開日 = 最新検証日と見なす)
- **公開後に実質的な内容更新** (引用の訂正・新規セクション追加・数値更新など) をした場合、frontmatter に `lastVerified: "YYYY-MM-DD"` を追加または更新する
- 軽微な修正(誤字脱字・書式調整)は `lastVerified` を更新しない
- `lastVerified` が `date` と異なる場合、コラム・戦略ページのヘッダーに「最終更新 YYYY-MM-DD」として表示される

## 5. 「分からない」と言える勇気

研究で確かめられていない場合、**「分からない」「実証されていない」と書く** ことが、エビデンスベースのサイトとしての誠実さです。沈黙より無理な数値を書く方が読者を裏切ります。

Hattie の有名な言葉:
> "Almost everything works in education, but the question is what works best."

しかし、**「working」と言える根拠が無いのに数字を書くのは『works』と主張するのと同じこと** です。注意してください。

## 6. 過去の失敗事例(教訓として)

### 事例 1: 朝食欠食

- 当初: `monthsGained: 3`(根拠なし)
- 検証後: `monthsGained: 1`、注記で「Adolphus 2016 は subtle と明記、月数換算なし」
- **教訓**: 「相関が強いから効果も大きいだろう」と推測してはいけない。家庭背景の交絡を統制した RCT が必要

### 事例 2: デジタル教科書(同上)

- 当初: `monthsGained: 1`(根拠なし)
- 検証後: `monthsGained: 0`、注記で「Delgado 2018 は g=-0.21(紙が優位)」
- **教訓**: 「新しいテクノロジー = 効果がある」と思い込まない。むしろ読解では負の効果が報告されている

### 事例 3: King et al. 2005 のジャーナル誤記

- 当初: 「Behavior Modification, 29(2), 281-290」
- 検証後: 「Behavioral Disorders, 30(3), 241-257」
- **教訓**: ジャーナル名は記憶ではなく検索で確認する。似た名前(Behavior Modification ↔ Behavioral Disorders)を取り違えやすい

### 事例 4: フィンランド PISA 2018 スコアの誤記

- 当初: 「507」
- 検証後: 「503」(さらに、最高点は 2003 の 544 ではなく 2006 の 548 だった)
- **教訓**: トレンドは正しくても具体値は記憶ベースで誤る。表に載せる数値は必ず一次資料で確認

## 8. 参考資料セクションの書式

コラム・戦略ページの末尾に置く参考資料リストは、**見出し・項目形式を統一** する。

### 見出し

```markdown
## 参考資料
```

他の揺れ(「参考」「参考文献」「主な参考資料」「参考データ・文献」「参考記事」等)は使わない。

### 項目形式

媒体別に以下のテンプレートを使う。

**学術論文(英文):**
```markdown
- 著者姓, F., et al. (YYYY). [Title](URL). *Journal*, Vol(Issue), Pages. — 一言補足
```

**日本語論文・寄稿:**
```markdown
- 著者名 (YYYY). [タイトル](URL). 媒体名. — 一言補足
```

**政府・行政資料:**
```markdown
- 機関名 (YYYY). [タイトル](URL). — 一言補足
```

**メディア記事:**
```markdown
- [記事タイトル](URL). 媒体名, YYYY-MM-DD. — 一言補足
```

**書籍:**
```markdown
- 『[書名 — 副題](URL)』 著者名 (YYYY), 出版社. — 一言補足
```

一般書は、読者(教員)が書店で探す感覚と揃えるため、**書名を先頭に** 置き、二重山括弧『』で包む。About ページの参考書籍一覧と同一書式。副題がある場合は全角ダッシュ「— 」で接続する。学術論文・政府資料・メディア記事は上記どおり「著者先頭」のまま維持する(学術慣例と衝突しないため)。

### 共通原則

- タイトルを **インラインリンクで包む** (`[タイトル](URL)`)
- 年号は著者の直後に `(YYYY)` で配置(**書籍は例外**: 書名を先頭に置き、著者名のあとに `(YYYY)` を置く)
- 一言補足は em-dash (`— `) で接続し、1〜2 文で記述
- 学術誌名はイタリック(`*...*`)で表記
- 書籍のタイトルは二重山括弧(`『 ... 』`)で囲み、リンクは二重山括弧の内側のタイトル部分だけに付ける
- 内部リンク(同サイトの戦略・コラム)は参考資料セクションには含めず、本文中のインラインリンクで扱う

### 並び順の規則

参考資料セクション内のエントリは、次のカテゴリ順で配置する。読者(小学校教員)が **一次情報に最短で到達できる** ことを優先し、「日本 → 国際」「一次 → 二次」「新 → 古」の 3 軸を組み合わせた順序。

1. **日本の政府・公的機関の一次統計/調査** — 文科省・NIER・国立教育政策研究所・自治体等が発表する統計・実態調査
2. **日本の法律・政策文書** — 法律本文、中央教育審議会答申、閣議決定、学習指導要領等
3. **日本の政府通知・周辺公表資料** — 課長通知、ガイドライン、解説パンフレット等
4. **日本の業界団体・研究機関・メディアの発信** — 教頭会/PTA 等団体調査、教育新聞等の報道、日本の寄稿
5. **国際機関の発信** — OECD、EEF Teaching and Learning Toolkit、What Works Clearinghouse 等
6. **学術論文** — メタ分析 → 系統的レビュー → RCT → 観察研究の順、同じエビデンス階層内は **主題との直接性** を優先しつつ、同等なら年代の新しい順
7. **書籍** — 日本語書籍 → 外国語書籍の順、同カテゴリ内は主題との直接性順

### 並び順の特例

- **主題の中心数値の一次ソース**(例: 本コラムが「76 万件」「8.8%」を論じるなら、その数値の公的調査)は、カテゴリ 1 内でも先頭に置く
- **政策解説コラム**(法律そのものを論じるコラム等)では、カテゴリ 2(法律)とカテゴリ 3(解説)の順序を入れ替え、「解説 → 法律本文」の順にしてもよい
- **サブ見出しを使う参考資料セクション**(例: ### 報道・分析 / ### 市場データ)でも、各サブ見出し内は上記の 7 カテゴリ順を適用する

### 外部リンクの挙動

外部リンクは `astro.config.mjs` の `rehype-external-links` プラグインにより、自動的に `target="_blank" rel="noopener noreferrer"` が付与される。Markdown 執筆時は通常の `[text](URL)` 記法のみ書けばよい。

## 9. 表現の明瞭性(スタイルガイド)

本サイトの主読者は **日本の小学校教員** です。母語で読んで違和感なく理解でき、職員室での会話でそのまま話題にできる文章を目指す。翻訳調・過度な抽象名詞・ビジネス用語は、エビデンスサイトとしての真摯さを損なう。

### 避けるべきパターン

| 類型 | 例 | 代替 |
|---|---|---|
| 英語直訳 | メインストリーム / エビデンスベースト / コンテクスト | 通常学級 / エビデンスに基づく / 文脈・状況 |
| 翻訳調(英語の動詞→名詞化) | 〜を有する / 〜に資する / 〜を担保する | 〜がある / 〜につながる / 〜を保つ |
| 英語の repertoire 構文 | すでに持っている / レパートリーにある | 既に実践している / 身につけている |
| ビジネス用語 | 見える化 / 感度を上げる / KPI / ロードマップ / アジャイル | 可視化・具体化 / 意識を高める / 指標 / 計画 |
| 抽象名詞化(構造・設計) | 〜の実装 / 〜の構造は避け / 〜の設計にしておく | 〜を取り入れる / 〜にならないように / 〜の仕組みを整える |
| 不要な漢語 | 鑑みる / 然るべき / 依拠する / 俯瞰する | 考えると / 適切な / 基づく / 見渡すと |
| 過度な体言止め | 「〜の明示化。」「〜への配慮。」を連発 | 文を動詞で締め、読むリズムを作る |

### 推奨される姿勢

- 「**教員が職員室の会話で使う言葉か**」を執筆時の判断基準にする
- 固い表現は、一次ソースの直接引用に限定する(Rule 1.2 の忠実性と両立)
- 一文 40〜60 字を目安にする。複文の多用を避ける
- 「〜のように」「〜といった」「〜という」を使う時は、具体例を示せるか点検する
- 読み上げてみて、息継ぎが不自然になる箇所は書き直す

### 一次ソース特有の比喩・業界用語は要チェック

一次ソース(調査報告・教育新聞・業界団体の発信など)には、その文脈内部で通じる比喩的表現や自称的表現が混じっていることがある(例: 教頭会の調査における「職員室の担任」)。これらを **引用する際に検証なしで使うと、一般読者に通じない** 可能性がある。

**チェック手順:**

1. その表現が **文部科学省文書・学校教育法・一般的な教育用語集** で確立されているか確認する
2. 確立されていない場合、以下のいずれかで対処する:
   - 引用元の文脈として明示する(「〇〇は△△と表現している」)
   - 直接的な表現に置き換える(比喩を使わず事実を述べる)
   - 削除する(その比喩がなくても主張が成立するなら削る)
3. **「聞いたことがない専門用語」は、読者も同じく聞いたことがない可能性が高い** と考える

### 例外

- **公的文書・法律の直接引用** は原文の固さを保つ(Rule 1.2 / 1.8 と整合)
- **学術用語・専門用語** は必要に応じて初出で定義しつつ用いる(例: メタ分析・RCT・効果量)
- **用語集にある用語** は本文で一度だけ詳しく説明し、以後は簡潔に用いる

### 過去の失敗例(本サイト)

- 「**メインストリーム**の教員が**すでに持っている**、あるいは**容易に取り入れられる**指導」
  → 「**通常学級**の教員が日常の授業で**既に実践している**、あるいは**少し意識することで取り入れられる**指導」
  ([special-needs-8-8-percent.md](../src/content/columns/special-needs-8-8-percent.md), PR #42)
- 「**家庭環境の多様性への感度を上げる** 機会」
  → 「**家庭環境の多様性への意識を高める** 機会」
  ([joint-custody-school-impact.md](../src/content/columns/joint-custody-school-impact.md), PR #42)
- 「**双方の学力向上に資する**」→「**双方の学力向上につながる**」
- 「**『職員室の担任』と呼ばれる副校長・教頭の負担が膨張している**」→「**副校長・教頭の負担が膨張している**」
  ([teacher-shortage-evidence.md](../src/content/columns/teacher-shortage-evidence.md), PR #46). 「職員室の担任」は教頭会の調査や一部の業界ブログで使われる自称的表現であり、文科省文書や一般的な教育用語集には存在しない。引用先で通じても一般読者には通じない例
