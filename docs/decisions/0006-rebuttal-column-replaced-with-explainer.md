# 0006. 反論型コラムを解説型に差し替える(前提検証ルール)

- 状態: 採用
- 日付: 2026-04-25
- 関連 PR: #102(close), #103, #104

## 背景

ジグソー法の効果量についてコラムを追加しようとした際、当初は「YouTube など日本語圏のメディアで d=1.20 が誤って拡散している」ことを前提にした **反論型コラム** を PR #102 として起案した。

運営者のレビューで「そもそも YouTube で d=1.20 が拡散しているという前提自体が本当か、検証すべきではないか」と指摘を受け、WebSearch で再検証した結果:

- 日本語圏のメディアがジグソー法を取り上げる際の主流は「知識構成型ジグソー法」(三宅なほみ / CoREF)で、海外オリジナル(Aronson 1978)の偏見低減の文脈を紹介するものが多い
- d=1.20 という具体的な数値を日本のメディアが拡散していると示すソースは **0 件**

前提そのものが虚構であり、PR #102 は「存在しない言説への反論」になっていた。

## 検討した選択肢

- **A) PR #102 をそのままマージ(前提の弱さを軽い注記で逃げる)**
- **B) PR #102 をクローズし、知識構成型ジグソー法を中心とした解説型コラムに書き直す**
- **C) PR #102 をクローズし、ジグソー法のコラム自体を諦める**

## 決定

**B)** を採用。

- PR #102 をクローズ(マージしない)
- 代わりに PR #103 で `strategies/jigsaw-method.md` に「日本独自の系譜 — 知識構成型ジグソー法」節を追加(三宅なほみ / CoREF の文脈 + Vives 2025 を追記)
- PR #104 で新規コラム `columns/jigsaw-two-lineages.md` を追加。Aronson 1978(偏見低減)と三宅なほみ(知識構成型)の 2 系譜を対比する解説型構成
- 効果量の言及は最小限(d=1.20 と「69 studies (Vives)」のみ)

### 付随する運営ルール

このインシデントから、**執筆前に前提を一次検証する** ルールを明文化し、ADR 0003 の `edu-pre-write-verifier` エージェント導入(執筆前ゲート)につながった。

運営者のメモリに `feedback_evidence_content_writing.md` の「コラムの前提も一次検証する」項を追加。

## 帰結

### 良い帰結

- 存在しない言説への反論を公開する事態を回避
- 日本の読者にとって馴染み深い「知識構成型ジグソー法」の文脈を正面から扱えた
- 「執筆前の前提検証」を一般原則として定着させる契機になった(ADR 0003 / `edu-pre-write-verifier` 導入)

### トレードオフ

- 当初執筆した草稿は破棄された(運営者の労力が一部無駄になった)
- 反論型の切り口が有効な場面でも、前提検証のコストを先払いする運用に切り替わる

## 撤回 / 再検討の条件

- 将来、日本語圏のメディアで実際に d=1.20 が誤拡散する状況が発生し、ソースとして複数確認できた場合(そのときは反論型コラムを改めて検討する)
