# 0005. 本体は非営利維持、収益化は別ブランド SaaS に分離

- 状態: 採用
- 日付: 2026-04-19 / 2026-04-23

## 背景

EduEvidence JP と姉妹サイト EduWatch JP は、公的機関や研究コミュニティから教員層までを読者に含む。運営継続には何らかの維持費負担が必要。ただし「エビデンスを整理して公開する」本体サイトに広告 / アフィリエイト / 有料コンテンツを混ぜた場合、次の 2 つが同時に起きる:

- 読者の「中立性への信頼」が損なわれる
- スポンサーの利害で編集判断が歪むリスクが入り込む

一方で「完全に無償でボランティア継続」は個人運営の限界に到達しやすく、大型助成や法人化を目指すにしても、資金源の設計が必要だった。

## 検討した選択肢

- **A) 本体に広告 / アフィリエイトを導入**
- **B) 本体に有料会員 / Premium 機能を導入**
- **C) 本体は非営利のまま。収益化は別ブランド・別ドメイン・別リポジトリの SaaS で実施。寄付と助成金で本体を支える**
- **D) 助成金と寄付だけで永続運営を目指す(SaaS なし)**

## 決定

**C)** を採用。

### 本体(edu-evidence / edu-watch)

- 広告なし
- アフィリエイトなし
- 有料壁なし
- 寄付(GitHub Sponsors 等)と助成金で維持
- 将来は一般社団法人化を検討

### 別ブランド SaaS

- **別ドメイン・別リポジトリ・別ブランド**で構築(信頼と収益の混在を避ける)
- ターゲット: 教員の**業務支援**(授業の編集ではなく、通常業務の負担を減らす方向)
- edu-evidence / edu-watch で築いた信頼 → SaaS への利用者導入、というファネル分離戦略

### 資金ロードマップ(併記)

- Phase A: GitHub Sponsors 設置、About ページに運営方針明記
- Phase B(3〜6 ヶ月): 博報堂教育財団 / パナソニック教育財団 / 中央教育研究所等の個人応募可能な助成金
- Phase C(1〜2 年): 一般社団法人化検討、中〜大型助成(トヨタ財団 / ベネッセこども基金 / 三菱財団)に射程拡大
- Phase D(長期): 公益認定

## 帰結

### 良い帰結

- 本体の編集判断が商業利害から独立し続ける
- 読者との信頼関係を前提にした週次 / 月次の運用設計が成立する
- SaaS が失敗しても本体が収益化から切り離されているため、継続の意思決定に影響しない

### トレードオフ

- SaaS の立ち上げコストと別組織化の手間が運営者に降りかかる
- 短期的な資金はすべて寄付 / 助成金に依存し、獲得前は運営者の自己負担になる
- 本体と SaaS の間で「信頼の貸し借り」が発生するため、SaaS 側の品質や行動が本体に逆流しないよう配慮が必要

## 撤回 / 再検討の条件

- 助成金が 2 年連続で獲得できず、本体維持が困難になった場合
- 一般社団法人化後、定款上の目的と両立する形で本体収益化が可能になった場合
- 別ブランド SaaS 構想そのものが撤回された場合
